# Web Frontend Project

## Overview
This project is a simple frontend web application that allows users to:
- View stored data
- Add new data
- Modify existing data

The application uses HTML, JavaScript, and browser localStorage for data persistence.

---

## Features

### Navigation
- Home page with links to all sections
- Easy navigation between all pages

### View Data
- Displays all stored entries

### Add Data
- Form input to add new data
- Stores data using localStorage

### Modify Data
- Edit existing entries
- Save updates in real-time

---

## Technologies Used
- HTML5
- JavaScript
- LocalStorage
